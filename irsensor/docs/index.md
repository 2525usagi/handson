
# 「SORACOM x RaspberryPi ハンズオン  〜赤外線リモコン編〜」

## 必要物
- SORACOM Air
- USB ドングル
- Raspberry Pi
- PC(Mac、Windowsなど)
- SDカードリーダー/ライター (PC本体にSDスロットがある場合には不要)
- Micro SD カード (8GB以上が望ましい)  
 ※もし初代Raspberry Piを利用する場合には、通常サイズのSDカードを用意して下さい
- 赤外線受信モジュール(秋月電子通商で販売 OSRB38C9AA)
- 赤外線LED(秋月電子通商で販売 OSI4FU5111)
- Githubアカウント
- RubyがインストールされたホストPC
